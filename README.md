# Yerdle

A Ye-themed daily word puzzle. Based on [https://github.com/cwackerfuss/react-wordle].

[Try it out here](https://yerdle.bensonkitia.com)

## Build and run

### To Run Locally:

Clone the repository and perform the following command line actions:

```bash
$> cd yerdle
$> npm install
$> npm run start
```
